주요 이벤트 - mouse 이벤트

- mouseenter
  => 해당 태그의 영역에 들어올 때 발생한다.
    - 한번 영역에 들어오면 바꿀 일이 없다 싶을 때 사용.
- mouseover
  => 마우스 커서가 해당 태그 위에 놓일 때 발생한다.
  => 자식 태그에 있다가 다시 부모 태그 위로 커서가 이동할 때도 발생한다.

- mouseleave
  => 마우스 커서가 해당 태그의 영역에서 벗어날 때 발생한다.
- mouseout
  => 마우스 커서가 해당 태그의 보이는 영역에서 벗어날 때 마다 발생한다.
   - 자식으로 갈때도 안보이기 때문에 발생한다.

- mouseup
  => 마우스 버튼을 뗄 떼 발생한다.
- mousedown
  => 마우스 버튼을 누를 떼 발생한다.
- mousemove
  => 마우스 커서를 움직일 때 마다 발생한다.
