웹브라우저 내장(built-in) 객체 - window.screen

- window : 각각의 탭마다 제공된다. 해당 탭에 윈도우 정보를 다룰 때 사용하는 객체이다.
- window.location
  현재 탭에서 출려하는 페이지의 URL 정보를 다룰 때 사용하는 객체이다.
- window.history
  사용자가 페이지 이동한 정보를 다룰 때 사용하는 객체이다.
- window.navigator
  웹 브라우저 정보를 다룰 때 사용하는 객체이다.(현재 사용자가 어떤 브라우저를 쓰는지 등)
- window.screen
  페이지를 출력하는 화면정보를 다룰 때 사용하는 정보이다.
